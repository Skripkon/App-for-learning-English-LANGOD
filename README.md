# How to run a server (local):

```python3 ./Server/server.py``` from ~/App-for-learning-English-LANGOD directory

# Packages:

| Field         | Package               |
| ------------- | --------------------- |
| Graphics      | ```PyQt5```                 |
| Server        | ```requests```, ```tornado```     |
| SQL           | ```sqlite3```               |
| Audio         | ```playsound```, ```os```         |
| Pronunciation | ```gtts```                  |
| Definitions   | ```wordnet```               |
| Usage         | ```reverso_context_api```   |
